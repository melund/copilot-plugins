# Data Visualization

Data visualization plugin that guides AI agents in creating clear, compelling
charts.

## What it does

This plugin covers areas:

| Area | Examples |
|------|----------|
| Chart design | Choosing chart types, decluttering, color usage, annotation, storytelling |

## Skills

### `matplotlib-data-visualization`

Activated when creating visualizations that need to communicate a clear message.
Focuses on visualization design principles rather than matplotlib API syntax —
chart type selection, removing clutter, directing attention with color and
annotation, and structuring data narratives. Teaches an opinionated approach
based on *Storytelling with Data* principles.

## Prerequisites

- Python with `matplotlib` for visualization tasks
