# Typst

Typesetting plugin that helps AI agents create documents, presentations, and
scientific papers using Typst, a modern alternative to LaTeX.

## What it does

Typst compiles in milliseconds and uses a simpler markup syntax than LaTeX while
supporting the same output quality for academic and scientific content.

| Area | Examples |
|------|----------|
| Documents | Academic papers, theses, scientific reports |
| Math | Inline and display equations, symbols, matrices |
| Tables | Complex layouts, merged cells, styled headers |
| Presentations | Slide decks via Polylux or Touying packages |
| Scripting | Variables, loops, conditionals, custom functions |
| Styling | Page setup, fonts, show/set rules, templates |

## Skills

### `typst`

Activated when creating documents with Typst, typesetting scientific content,
writing mathematical equations, or when the user mentions Typst or document
formatting. Includes reference material for Typst's three syntactical modes
(markup, math, code) and popular packages: CeTZ (graphics), Polylux/Touying
(presentations), Codly (code listings), Lilaq (plots), and others.

## Prerequisites

- [Typst](https://typst.app/) CLI or the Typst web editor
