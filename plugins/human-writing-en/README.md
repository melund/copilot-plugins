# Human Writing (English)

Writing style plugin that helps AI agents produce English text that reads as
human-authored rather than AI-generated.

## What it does

LLM-generated text has identifiable patterns — inflated vocabulary, vague hedging,
formulaic structure. This plugin teaches agents to avoid those patterns and write
prose that a competent human writer would produce.

| Area | Examples |
|------|----------|
| Vocabulary | Banned words list (e.g., "moreover", "leverage", "delve") |
| Sentence structure | Short sentences, active voice, concrete nouns |
| Tone | Direct statements, no excessive hedging or enthusiasm |
| Formatting | Varied paragraph length, natural flow |

## Skills

### `human-writing-en`

Activated when writing prose, documentation, explanations, blog posts, emails,
articles, reports, or any English text that should sound natural. Also activated
when editing text to remove AI-sounding patterns, or when the user mentions
writing style, tone, or voice.
